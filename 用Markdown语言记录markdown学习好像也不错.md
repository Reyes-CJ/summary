### 2.字体

```python
'''
#学习了加粗
#加粗：左右使用两个**,如右边引号内部分"**加粗**"
#高亮
#高亮：使用左右两个=，如右边"==高亮=="
#删除线:左右两个~~，这样："~~删除线~~"
#斜体:左右各一个*，记得和加粗做出区分，例："*斜体*"
'''

astr = '6'*100
for i in range(100):
	print('XY,DY,HY,TXDY',astr,'\r\n')
```

#### 来看看效果：

**加粗**

==高亮还不知道为啥不行==

~~删除线啊~~

***加粗倾斜***
